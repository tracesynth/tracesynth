Tests produced by litmus for architecture RISCV on linux 

COMPILING
  with command 'make [-j N]' or 'sh comp.sh'

RUNNING ALL TESTS
  with command 'sh run.sh'. Test result on standard output.

RUNNING ONE TEST
  Tests are .exe files, for instance SB.exe, run it by './SB.exe'

RUNNING OPTIONS
  Main options to the run.sh script and to .exe files:
  -v     be verbose (can be repeated).
  -a <n> number of (logical) processors available, default 4.
  -s <n> one run operates on arrays of size <n>, default 2000.
  -r <n> number of runs, default 20000.

  For more options see for instance './SB.exe -help' and litmus documentation
  <http://diy.inria.fr/doc/litmus.html>
